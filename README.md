# Real-time Weather App

A beautiful, real-time weather application built with React, Vite, and the Open-Meteo API.

## Features
- **Real-time Data**: Fetches the latest available weather data for the current hour.
- **Global Search**: Search for any city worldwide.
- **Glassmorphism Design**: Modern, aesthetically pleasing UI with animated backgrounds.
- **Responsive**: Works on all devices.

## How to Run

1.  Navigate to the project directory:
    ```bash
    cd realtime-weather-app
    ```

2.  Install dependencies (if not already installed):
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser at the Local URL shown (usually `http://localhost:5173/`).

## API Used
- [Open-Meteo](https://open-meteo.com/) (Free, Open-Source Weather API)
